# 차량 기지 알고리즘
- 중위 표기식을 후위 표기식으로 바꾸는 알고리즘

### 조건
1. 연사자만 스택에 저장
2. 연산자 > stk.top() => ret에 추가
   연산자 < stk.top() => 연산자 스택.push()
3. 괄호가 있는 경우
  - 여는 괄호가 나오면 닫는 괄호가 나올 때까지 연산자 스택에서 연산자를 꺼낸다.

### 예시
중위 표현식 => $3+2*3-4$
|연산자/피연산자|연산자 스택|결과|
|:-----------:|:--------:|---:|
|3| |3|
|+|+|3|
|2|+|32|
|*|+*|32|
|3|+*|323|
|-|+|323*|
|-|-|323*+|
|4|-|323*+4|
| | |323*+4-|
