# YOLOv8 데이터셋 준비 가이드

이 문서에서는 YOLOv8을 사용하고 데이터셋을 준비하는 과정을 설명하려고 합니다.

## 1. 데이터셋 구조 설정

YOLOv8은 아래와 같은 폴더 구조를 갖춰야합니다.

- images/
    - train/
    - val/
- labels/
    -train/
    -val/

```bash
tip!
#폴더를 옮기거나 복사할 때는 cli를 사용해서 옮기도록 한다.(속도가 빠름)
#이미지가 너무 많아서 이동이 안될 경우는 다음과 같은 명령어를 사용한다.

find bdd100k/images/100k/train -type f -exec mv {} . \;
```
명령어를 설명하면 다음과 같다.
**find** : 파일 및 디렉토리를 검색하는 명령어
**bdd100k/images/100k/train** : 검색할 디렉토리 경로
**-type f** : 파일 유형을 지정하는 옵션
**-exec** : find 명령어와 함께 사용하는 옵션으로 찾은 각 파일에 대해 지정한 명령을 실행합니다.
**mv** : 파일을 이동하는 명령어
**{}** : find가 찾은 각 파일의 경로를 의미합니다. {}는 -exec와 함께 사용하여 현재 파일의 경로를 받아오는 역할을 합니다.
**.** : 파일을 이동할 대상 경로입니다.
**\;** : -exec 명령어의 종료를 나타내는 기호입니다.
해당 명령어는 각 파일에 대해 실행한다는 의미이다.


## 2. 이미지 및 라벨 파일 준비

1. 이미지
    - 이미지 파일은 '.jpg' 또는 '.png' 형식을 사용해야 합니다.
    - 해상도를 일정하게 맞추는 것이 좋습니다.

2. 라벨
    - 라벨 파일과 이미지 파일명은 동일한 이름을 가지며, '.txt' 확장자를 사용합니다.
    - 라벨 파일 포맷은 '[클래스 번호] [중심 x] [중심 y] [너비] [높이] ' (0 ~ 1사이의 상대 좌표)
    - ex : 0 0.5 0.5 0.2 0.2 
        - 클래스 0
        - 중심 (0.5 0.5)
        - 너비 0.2
        - 높이 0.2

### 필요한 정보 추출하기
보통 필요한 정보를 추출하기 위해서는 파이썬 코드를 작성해서 내가 원하는 값을 라벨 데이터로부터 추출해야 합니다.
과정을 설명하면 다음과 같습니다.
1. 이미지 파일명과 라벨 파일 명이 동일한지 확인합니다.
2. 이미지에서 필요한 정보들을 라벨에서 얻어온다. (예를 들어 맑은 날씨와 악천을 구분하고 싶다면, 라벨에서 weather을 찾아 이미지와 라벨을 분류한다.)

## 3. 클래스 정의 파일 작성 (선택 사항)

'data.yaml' 파일을 생성하여 데이터셋의 경로와 클래스 정보를 정의합니다.

파일 내용은 다음과 같습니다.
```yaml
# data.yaml
train: ../datasets/images/train
val: ../datasets/images/val
nc: 2  # 클래스 수
names: ['class1', 'class2']  # 클래스 이름
```
